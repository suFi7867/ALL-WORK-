b tag

The b tag in HTML is used to specify the bold text without any extra importance. The text is written within b tag display in bold size.

i tag

The i tag in HTML is used to display the content in italic style.

u tag

The u tag in HTML is used to display the content in underlined character or word.

select tag

The select element is used to create a drop-down list.

button tag

The button tag defines a clickable button.
